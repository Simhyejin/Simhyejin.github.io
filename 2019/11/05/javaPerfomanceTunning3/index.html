StringBuffer

  • 스레드 세이프해야 할 경우 사용

StringBuilder

  • 스레드 세이프하지 않아도 될 때 사용

JDK 5.0 이상

JDK 5.0 이상부터는 컴파일러가 String으로 문자열을 더할 때 StringBuilder로 변환된다.

그래도
String 클래스를 쓰는 대신 스레드와 관련이 있으면 StringBuffer를
스레드 안전 여부와 상관이 없으면 StringBuilder를 사용을 권장!
